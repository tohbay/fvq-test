# FVQ Test
This is a Frontend Development assessment.

## Technologies used
- React - Vite 
- Vanilla css for minimalistic approach
- Vercel for deployment

## Approach
- Mobile-first approach design was used in building this project
- Minimal basic responsiveness was achieved using media-queries
- No external libraries were used as the project is small

## Observations
- There were observed typos on the Figma design.
- Project was built correcting the typos. It could be a test for attention to details.
- Form validation was intentionally skipped. However, there could be two approaches:
  a. show error message on each field not filled
  b. disable the ```Continue``` button if any field is not field.
  These are subject to project decisions and requests.
